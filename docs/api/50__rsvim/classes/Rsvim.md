The `Rsvim` global object, it contains multiple sub fields:

- `Rsvim.opt`: Global editor options.

## Example

```javascript
// Create a variable alias to 'Rsvim'.
const vim = Rsvim;
```

## Properties

| Property | Modifier | Type |
| ------ | ------ | ------ |
| `opt` | `readonly` | [`RsvimOpt`](RsvimOpt.md) |
