# Installation

## Download

You can download pre-built binary from GitHub release page:

<Tabs>
  <TabItem value="linux" label="linux" default>
    ```bash
    curl -fsSL https://github.com/rsvim/rsvim/releases/latest/download/rsvim-x86_64-unknown-linux-gnu.tar.gz -o rsvim.tar.gz
    tar zxf rsvim.tar.gz -C rsvim
    ```
  </TabItem>
  <TabItem value="mac" label="mac">
    ```bash
    curl -fsSL https://github.com/rsvim/rsvim/releases/latest/download/rsvim-aarch64-apple-darwin.tar.gz -o rsvim.tar.gz
    tar zxf rsvim.tar.gz -C rsvim
    ```
  </TabItem>
  <TabItem value="windows" label="windows">
    ```powershell
    Invoke-WebRequest -Uri https://github.com/rsvim/rsvim/releases/latest/download/rsvim-x86_64-pc-windows-msvc.tar.gz -OutFile rsvim.tar.gz
    ```
  </TabItem>
</Tabs>

## Build From Source

You can also build with `cargo`:

```bash
cargo install --locked rsvim
```

For the latest updates, build with `main` branch:

```bash
cargo install --locked rsvim --git https://github.com/rsvim/rsvim.git --branch main
```
